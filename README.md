O JogoGreenZone é um jogo de plataforma 2D desenvolvido em Python utilizando a biblioteca Pygame.

É um projeto focado em demonstrar e estudar os fundamentos da criação de jogos 2D, com ênfase nas seguintes funcionalidades principais:
Animação de Personagem: Uso de sprites para criar animações de movimento, corrida e pulo.
Mecânica de Plataforma: Implementação de física básica (gravidade e pulo) e movimentação horizontal.
Sistema de Tiles: Estrutura de mapa baseada em matriz para construção eficiente do cenário.
Detecção de Colisão: Lógica para interação correta entre o personagem e os blocos do nível.

O repositório do GitHub contém o código-fonte, a estrutura de pastas e instruções detalhadas sobre como instalar o Pygame e executar o jogo.
